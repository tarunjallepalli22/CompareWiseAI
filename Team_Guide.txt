🤝 Team Guide: CompareWise AI"Procurement intelligence, without the privacy headache."Hey team! This is our new platform for comparing vendor proposals. This guide will help you understand what we've built, how it works, and how to get it running on your own machine.---🌟 What is CompareWise AI?Think of this as a "Super-Powered Assistant" for our procurement team. Instead of manually reading five 20-page PDFs and trying to remember who offered the best price or which one had that weird legal clause, you just drag them into this app.The AI reads them all instantly and gives you a side-by-side table and a recommendation.Why this is better than ChatGPT:100% Private: We are not sending our vendor data to the cloud. Everything stays on your laptop.Fast: It handles multiple files at once.Structured: It doesn't just "chat"—it gives us a real table we can use for reports.---🛠️ How it Works (The Simple Version)You Upload: You drop PDFs, Word docs, or Excel sheets into the dashboard.Ollama (Our Local Brain): We use a tool called Ollama that runs a smart AI model (`llama3.2`) right on your computer. It reads the text and picks out:Total CostDelivery TimelineSLA (Guarantees)Risk Factors (The "Fine Print")The Result: The app builds a comparison table and writes an "AI Recommendation" on who we should pick.---🚀 How to Run it LocallyIf you want to run this code on your machine, follow these steps:1. The "Brain" (Ollama)Download and install Ollama from ollama.com.Open your terminal/command prompt and run: `ollama run llama3.2`Keep this running in the background.2. The CodeOpen the `CompareWiseAI` folder.If you haven't yet, install the requirements:    ```bash    pip install -r requirements.txt    ```Start the server:    ```bash    python main.py    ```Go to http://127.0.0.1:8000 in your browser.---📂 Folder Tour: Where is everything?`main.py`: The "Backend". This handles the file reading and talks to the AI.`static/`: The "Frontend". This is the website you see (HTML, CSS, JS).`SAMPLES(PDF)/`: If you don't have a proposal to test, use the files in here!`requirements.txt`: The list of Python tools we need to make this work.---💡 Pro-Tips for the TeamCaching: If you upload the same file twice, it will load instantly. The app "remembers" files it has already analyzed.Light/Dark Mode: Use the toggle in the top right—whatever is easier on your eyes!Adding New Vendors: You can add our own "Top Recommended" vendors by putting their PDFs in `static/best_proposals/`.---Questions?
