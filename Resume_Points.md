## 📋 Full Project Description (For Portfolios / Detailed Resumes)

**CompareWise AI — Offline-First Vendor Intelligence Platform**
* **Role:** Lead Full-Stack & AI Engineer
* **Description:** Developed an offline-first, enterprise-ready procurement analysis tool designed to eliminate manual vendor proposal comparisons. By configuring a local LLM runner (Llama 3.2) on Ollama, the application extracts critical commercial data points (Total Cost, Timeline, SLA, hidden Red Flags/Risks) without leaking sensitive corporate data to external cloud providers. Optimized execution speeds through Python async threading and MD5-based document caching.
